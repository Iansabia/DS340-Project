## Decisions Made
- **Ensemble averaging:** LR + XGBoost predictions averaged for entry signal (simple, robust)
- **Bar interval gating:** Strategy only collects prices for pairs whose tier interval has elapsed since last collection, matching the cron architecture from CONTEXT.md
- **Snapshot bar features:** Uses build_snapshot_bar() with zero volume/trade fields and fills missing derived features with 0.0
- **Dry-run mode:** Computes full cycle (classify, collect, predict) but skips DB writes and bar persistence
