<!-- Position state schema from CONTEXT.md: -->
```python
@dataclass
class Position:
    pair_id: str
    kalshi_ticker: str
    direction: str              # "short_spread" or "long_spread"
    entry_spread: float         # spread at entry
    entry_time: str             # ISO 8601 UTC
    entry_kalshi_price: float
    entry_poly_price: float
    current_spread: float
    bars_held: int
    max_spread_since_entry: float
    min_spread_since_entry: float
    unrealized_pnl: float
    stop_loss: float            # computed at entry: entry_spread * 1.5
    take_profit: float          # fixed: 0.10
    trailing_stop_offset: float # fixed: 0.15
    tier: str                   # from ContractClassifier
    resolution_date: str | None # ISO 8601 or None
```

<!-- Exit rules from CONTEXT.md (exact thresholds): -->
1. Take Profit:     spread < 10pp (0.10)
2. Stop Loss:       spread > entry_spread * 1.5
3. Trailing Stop:   spread > min_spread_since_entry + 15pp (0.15)
4. Momentum Exit:   4 consecutive bars where spread moved against direction
5. Time Stop:       24h for ACTIVE_DAY/SPRINT, 72h for ACTIVE_WEEK, 168h for WARMING/MONITORING
6. Resolution Exit: < 2 hours to resolution

<!-- ExitReason enum: -->
TAKE_PROFIT, STOP_LOSS, TRAILING_STOP, MOMENTUM, TIME_STOP, RESOLUTION_EXIT, MANUAL
```

  <name>Task 2: Six exit rules with check_exits method</name>
  <files>src/live/position_manager.py, tests/test_position_manager.py</files>
  <behavior>
    - check_exits(pair_id, now, spread_history=None) -> ExitReason | None
    - Take Profit: triggers when abs(current_spread) < 0.10 for short_spread positions (spread converged)
    - Stop Loss: triggers when current_spread > entry_spread * 1.5 (spread widened)
    - Trailing Stop: triggers when current_spread > min_spread_since_entry + 0.15 (bounced from best)
    - Momentum Exit: triggers when spread_history shows 4 consecutive bars moving against position direction. spread_history is list of last N spreads. "Against direction" for short_spread = spread increased for 4 bars in a row
    - Time Stop: triggers based on tier. ACTIVE_DAY/SPRINT: bars_held * bar_interval > 24*3600. ACTIVE_WEEK: > 72*3600. WARMING/MONITORING: > 168*3600. Uses tier and bar_interval from position
    - Resolution Exit: triggers when hours_to_resolution < 2 (computed from resolution_date and now)
    - check_all_exits(now, spread_histories) -> list[(pair_id, ExitReason)] checks all open positions
    - First exit rule that fires wins (priority order: RESOLUTION > STOP_LOSS > TAKE_PROFIT > TRAILING_STOP > MOMENTUM > TIME_STOP)
  </behavior>
