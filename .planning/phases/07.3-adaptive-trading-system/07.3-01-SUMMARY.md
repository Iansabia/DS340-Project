## Decisions Made

- Added dedicated regex for primary-style tickers (KXFLPRIMARY-23D26-JMOS) where year is embedded after a D prefix
- Tier boundaries use strict less-than: exactly 7 days = WEEKLY, exactly 30 = MONTHLY, exactly 90 = QUARTERLY
