    ```python
    def save(self, path: str | Path) -> None:
        """Save trained model to pickle file."""
        import pickle
        path = Path(path)
        path.parent.mkdir(parents=True, exist_ok=True)
        with open(path, 'wb') as f:
            pickle.dump(self, f)

    @classmethod
    def load(cls, path: str | Path) -> "BasePredictor":
        """Load a trained model from pickle file."""
        import pickle
        with open(path, 'rb') as f:
            model = pickle.load(f)
        if not isinstance(model, BasePredictor):
            raise TypeError(f"Loaded object is {type(model)}, not BasePredictor")
        return model
    ```

    **Constructor:**
    ```python
    def __init__(self,
                 live_dir: Path = Path("data/live"),
                 model_dir: Path = Path("models/deployed"),
                 min_price: float = 0.10,
                 min_spread: float = 0.30,
                 prediction_threshold: float = 0.02):
    ```
    - Loads ContractClassifier
    - Loads PositionManager(state_path=live_dir/"positions.json", history_path=live_dir/"position_history.jsonl")
    - Loads LR + XGBoost from model_dir pickles via BasePredictor.load()
    - Stores filter parameters

    **run_cycle(self) -> dict:**
    Full trading cycle:

    1. **Classify**: Load active_matches.json, run classify_all_pairs(now=utcnow)
    2. **Collect prices**: Use LiveCollector to fetch kalshi_prices and poly_prices
    3. **Update open positions**: For each open position, look up current prices, compute current_spread, call pm.update_position(pair_id, current_spread)
    4. **Check exits**: Build spread_histories from bars.parquet (last 5 bars per pair). Call pm.check_all_exits(now, spread_histories). For each triggered exit, call pm.close_position(pair_id, reason, current_spread). Log exit to console.
    5. **Check entries**: For each pair NOT in an open position:
       - Skip if tier is NO_TRADE or UNKNOWN
       - Skip if max(kalshi_price, poly_price) < min_price
       - Compute spread = kalshi_price - poly_price
       - Skip if abs(spread) < min_spread
       - Run LR + XGBoost predictions on the pair's features (build feature vector from current prices, use build_snapshot_bar from collector)
       - Average the two model predictions
       - Skip if abs(avg_prediction) < prediction_threshold
       - Determine direction: "short_spread" if spread > 0 (bet spread narrows), "long_spread" if spread < 0
       - Call pm.open_position(pair_id, kalshi_ticker, direction, abs(spread), kalshi_price, poly_price, tier, resolution_date)
       - Log entry to console
    6. **Save**: pm.save()
    7. **Return summary**: {"entries": N, "exits": N, "open_positions": N, "pairs_checked": N, "prices_fetched": N}

    **CLI block (if __name__ == "__main__"):**
    - `python -m src.live.strategy --cycle` runs one full cycle
    - `python -m src.live.strategy --status` prints current open positions and their unrealized P&L
    - `python -m src.live.strategy --export-models` trains and saves LR + XGBoost pickles
    - `--live-dir`, `--model-dir` optional path overrides
    - `--dry-run` flag: runs cycle but doesn't save positions (for testing)

    **Important implementation notes:**
    - For entry feature vectors, create a minimal feature dict from the two prices using LiveCollector.build_snapshot_bar, then compute derived features. The model only needs the 31 feature columns.
    - For spread_histories, read the last 5 bars for each pair from bars.parquet, extract the spread column as a list.
    - Handle missing prices gracefully: skip pairs where either Kalshi or Polymarket price is None.
    - Log all decisions: "ENTRY: live_0042 short_spread @ 0.52 (LR=0.03, XGB=0.04)" and "EXIT: live_0042 TAKE_PROFIT @ 0.08 P&L=+0.44 held=12bars"
