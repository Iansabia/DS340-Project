    ```python
    def save(self, path: str | Path) -> None:
        """Save trained model to pickle file."""
        import pickle
        from pathlib import Path
        path = Path(path)
        path.parent.mkdir(parents=True, exist_ok=True)
        with open(path, 'wb') as f:
            pickle.dump(self, f)

    @classmethod
    def load(cls, path: str | Path) -> "BasePredictor":
        """Load a trained model from pickle file."""
        import pickle
        from pathlib import Path
        with open(path, 'rb') as f:
            model = pickle.load(f)
        if not isinstance(model, BasePredictor):
            raise TypeError(f"Loaded object is {type(model)}, not BasePredictor")
        return model
    ```

    **src/live/strategy.py -- TradingStrategy class:**

    Constructor:
    ```python
    def __init__(self,
                 live_dir: Path = Path("data/live"),
                 model_dir: Path = Path("models/deployed"),
                 min_price: float = 0.10,
                 min_spread: float = 0.30,
                 prediction_threshold: float = 0.02):
    ```
    - Instantiates ContractClassifier
    - Instantiates PositionManager(db_path=live_dir/"positions.db", history_jsonl_path=live_dir/"position_history.jsonl")
    - Loads LR + XGBoost from model_dir via BasePredictor.load()
    - Loads feature_columns from model_dir/feature_columns.json
    - Loads active_matches.json
    - Stores filter thresholds

    **run_cycle(self, dry_run: bool = False) -> dict:**
    Full trading cycle (runs in < 60 seconds):

    1. **Classify**: classify_all_pairs(active_matches, now=utcnow, use_api=False)
       (use cached resolution dates from pair_classifications.json, no API calls during cycle)
    2. **Determine collection set**: For each pair, check if bar_interval has elapsed since last collection.
       Read last collection timestamp per pair from bars.parquet. Skip pairs where elapsed < bar_interval.
       This is how the 5-minute cron handles all tiers: 4h-tier pairs only get new bars every 4 hours.
    3. **Collect prices**: Use LiveCollector to fetch kalshi_prices and poly_prices for the collection set only.
    4. **Update open positions**: For each open position, look up current prices by kalshi_ticker and poly_id.
       Compute current_spread = kalshi_price - poly_price. Call pm.update_position(pair_id, current_spread).
       Skip positions where price data is unavailable.
    5. **Check exits**: Call pm.check_all_exits(now). For each triggered exit:
       Call pm.close_position(pair_id, reason, current_spread, now_iso).
       Log: "EXIT: {pair_id} {reason.name} @ {exit_spread:.4f} P&L={realized_pnl:+.4f} held={bars_held}bars"
    6. **Check entries**: For each pair in the collection set that is NOT in an open position:
       - Skip if tier is UNKNOWN
       - Skip if max(kalshi_price, poly_price) < min_price
       - Compute spread = kalshi_price - poly_price
       - Skip if abs(spread) < min_spread
       - Build feature vector: use LiveCollector.build_snapshot_bar to create a bar dict,
         wrap in DataFrame, compute derived features via compute_derived_features(),
         extract feature_columns
       - Run LR.predict(features) and XGBoost.predict(features)
       - Average predictions: avg_pred = (lr_pred + xgb_pred) / 2
       - Skip if abs(avg_pred) < prediction_threshold
       - Direction: "short_spread" if spread > 0 (bet spread narrows), "long_spread" if spread < 0
       - tier_info = classifications[pair_id]
       - pm.open_position(pair_id, kalshi_ticker, direction, abs(spread),
                          kalshi_price, poly_price, tier_info["tier"],
                          tier_info["bar_interval_seconds"], tier_info.get("resolution_date"))
       - Log: "ENTRY: {pair_id} {direction} @ {spread:.4f} (LR={lr_pred:.4f}, XGB={xgb_pred:.4f})"
    7. **Save state**: (SQLite auto-saves, but flush any pending commits)
    8. **Build bars**: For collected pairs, build and append bars to bars.parquet (for future lookback)
    9. **Return summary**: {"timestamp": now_iso, "entries": N, "exits": N,
       "open_positions": len(pm.get_open_positions()), "pairs_checked": N,
       "prices_fetched": N, "exit_details": [...], "entry_details": [...]}

    If dry_run=True: compute everything but don't save positions or append bars.
