Trade decision logic (matching evaluate()):
- `prediction = model.predict(X)`
- Trade when `abs(prediction) > threshold` (threshold=0.02)
- Direction: buy Kalshi/sell Polymarket when prediction > 0 (expects spread to widen), opposite when prediction < 0

Paper trade JSONL format (one line per trade event):
```json
{
  "timestamp": 1775500000,
  "collection_time": "2026-04-05T12:00:00Z",
  "pair_id": "kxbtc26feb0617b-0x0356fe1e",
  "model": "XGBoost",
  "prediction": 0.035,
  "direction": "long_spread",
  "threshold": 0.02,
  "kalshi_price": 0.55,
  "polymarket_price": 0.48,
  "spread": 0.07,
  "trade": true
}
```

When a model's prediction is below threshold, log with `"trade": false` (skip) to track total signal volume.

**`load_trade_log(path: Path) -> list[dict]`:**
- Read `data/live/paper_trades.jsonl` line by line
- Parse each line as JSON
- Return list of trade dicts
- If file doesn't exist, return empty list

**`compute_paper_pnl(trades: list[dict], resolution_data: pd.DataFrame | None = None) -> dict[str, dict]`:**
- Group trades by model name
- For each model:
  - Count total signals (all entries)
  - Count actual trades (where `trade == True`)
  - If resolution_data is provided (future bars showing what actually happened to the spread):
    - Match each trade to its resolution bar (same pair_id, next timestamp)
    - actual_change = resolution_spread - trade_spread
    - P&L = direction * actual_change (positive if prediction was right)
    - Compute win_rate = wins / trades, cumulative_pnl = sum(individual P&L)
  - If no resolution data: report trade count, signal count, avg prediction magnitude
- Return `{model_name: {signals, trades, wins, losses, cumulative_pnl, win_rate, avg_prediction}}`

**`print_dashboard(live_dir: Path = Path("data/live"))`:**
- Load trade log from paper_trades.jsonl
- Load bars.parquet if it exists (for resolution matching)
