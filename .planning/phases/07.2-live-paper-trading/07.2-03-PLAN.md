**`load_combined_data() -> tuple[pd.DataFrame, pd.DataFrame]`:**
- Load original `train.parquet` from data_dir
- Load `bars.parquet` from live_dir (if exists; if not, log warning and use original only)
- Concatenate: `combined_train = pd.concat([original_train, live_bars], ignore_index=True)`
- Sort by (pair_id, time_idx)
- Deduplicate on (pair_id, timestamp) keeping first
- Load original `test.parquet` from data_dir (test set stays fixed for fair comparison)
- Return (combined_train, test)
- Log: "Combined training data: {n_original} original + {n_live} live = {n_combined} total"

**`evaluate_baseline(test: pd.DataFrame, feature_cols: list[str]) -> dict[str, dict]`:**
- Train all 8 models on ORIGINAL train.parquet only (no live data)
- Evaluate on test set
- Return `{model_name: {rmse, mae, directional_accuracy}}`
- This is the "before" measurement

Training follows the exact run_baselines.py pattern:
```python
# Build split
train_split = _build_split(original_train)
test_split = _build_split(test)
feature_cols = _feature_columns(train_split)

# Timestamps for panel-aware Sharpe

# Tier 1
X_train, y_train = prepare_xy(train_split, feature_cols)
X_test, y_test = prepare_xy(test_split, feature_cols)
for model in build_models(tier="1"):
    model.fit(X_train, y_train)
    metrics = model.evaluate(X_test, y_test, threshold=threshold, timestamps=test_timestamps)
    # Store {rmse, mae, directional_accuracy} from metrics

# Tier 2
X_train_seq, y_train_seq = prepare_xy_for_seq(train_split, feature_cols)
X_test_seq, y_test_seq = prepare_xy_for_seq(test_split, feature_cols)
for model in build_models(tier="2"):
    model.fit(X_train_seq, y_train_seq)
    metrics = model.evaluate(X_test_seq, y_test_seq, threshold=threshold, timestamps=test_timestamps)

# Tier 3
autoencoder = AnomalyDetectorAutoencoder(input_dim=len(feature_cols), random_state=42)
autoencoder.fit(train_split[feature_cols], feature_cols)
ppo_raw = PPORawPredictor(random_state=42, total_timesteps=100_000)
ppo_raw.fit(X_train_seq, y_train_seq)
ppo_filt = PPOFilteredPredictor(anomaly_detector=autoencoder, random_state=42, total_timesteps=100_000)
ppo_filt.fit(X_train_seq, y_train_seq)
# evaluate both on test set
```

**`train_and_evaluate_combined(combined_train: pd.DataFrame, test: pd.DataFrame) -> dict[str, dict]`:**
- Same as evaluate_baseline but trains on combined_train (original + live)
- Return `{model_name: {rmse, mae, directional_accuracy}}`
- This is the "after" measurement

**`compute_deltas(before: dict, after: dict) -> dict[str, dict]`:**
- For each model, compute delta = after - before for each metric
- For RMSE/MAE: negative delta = improvement (lower is better)
- For directional_accuracy: positive delta = improvement (higher is better)
- `improved = (delta_rmse < 0)` (primary metric)
- Return `{model_name: {before, after, delta, improved}}`

**`log_retrain_event(deltas: dict, n_original: int, n_live: int, n_combined: int, n_test: int)`:**
- Load existing retrain_log.json if it exists, else create new
- Append new event with timestamp, row counts, and per-model deltas
- Write back to data/live/retrain_log.json

**`run() -> dict`:**
