    BasePredictor ABC:
    - Has abstract method predict(X: pd.DataFrame) -> np.ndarray returning predicted spread values
    - Has abstract property name -> str
    - Has fit(X_train: pd.DataFrame, y_train: np.ndarray) -> self (default no-op for baselines)

    compute_regression_metrics(y_true, y_pred):
    - Test: RMSE of [1,2,3] vs [1,2,3] == 0.0
    - Test: MAE of [1,2,3] vs [2,3,4] == 1.0
    - Test: RMSE of [0,0,0] vs [1,1,1] == 1.0
    - Test: directional accuracy of [0.1, -0.1, 0.2] true vs [0.05, -0.2, 0.3] pred == 1.0 (all same sign)
    - Test: directional accuracy of [0.1, -0.1] true vs [-0.1, 0.1] pred == 0.0 (all wrong sign)
    - Test: returns dict with keys "rmse", "mae", "directional_accuracy"

    simulate_profit(predictions, actuals, threshold=0.02):
    - Strategy: trade when |predicted_spread| > threshold. Direction: if predicted spread > threshold, bet spread will close (buy underpriced platform). If predicted spread < -threshold, bet spread will widen (sell overpriced platform). Profit per trade = actual_spread_change * position_direction.
    - Test: all predictions below threshold -> 0 trades, P&L = 0, win_rate = 0, sharpe = 0
    - Test: single profitable trade (predict +0.05, actual closes +0.03) -> positive P&L
    - Test: mixed profitable and losing trades -> correct win_rate (# profitable / # total)
    - Test: Sharpe ratio = mean(returns) / std(returns) * sqrt(252) (annualized, daily if hourly then adjust)
    - Test: returns dict with keys "total_pnl", "num_trades", "win_rate", "sharpe_ratio", "pnl_series"
  </behavior>

       - Abstract method: predict(X: pd.DataFrame) -> np.ndarray (returns array of predicted spread values, one per row)
       - Abstract property: name -> str (human-readable model name for results tables)
       - Concrete method: fit(X_train: pd.DataFrame, y_train: np.ndarray) -> "BasePredictor" (default no-op, overridden by trainable models). Returns self for chaining.

    3. Create src/evaluation/__init__.py (empty package init with docstring "Evaluation framework for spread prediction models").
    4. Create src/evaluation/metrics.py with:
       - compute_regression_metrics(y_true: np.ndarray, y_pred: np.ndarray) -> dict
       - Returns {"rmse": float, "mae": float, "directional_accuracy": float}
       - RMSE: sqrt(mean((y_true - y_pred)^2))
       - MAE: mean(|y_true - y_pred|)
       - Directional accuracy: fraction of samples where sign(y_true) == sign(y_pred). Exclude samples where y_true == 0 (no direction to predict). Use np.sign.

    5. Create src/evaluation/profit_sim.py with:
       - simulate_profit(predictions: np.ndarray, actuals: np.ndarray, threshold: float = 0.02) -> dict
       - Trading logic: for each timestep i, if |predictions[i]| > threshold, enter trade. Position direction = sign(predictions[i]). Profit = actuals[i] * sign(predictions[i]). (actuals represents the actual spread change over the prediction horizon.)
       - Returns: {"total_pnl": float, "num_trades": int, "win_rate": float, "sharpe_ratio": float, "pnl_series": list[float]}
       - win_rate = count(pnl > 0) / num_trades. If num_trades == 0, win_rate = 0.0.
       - sharpe_ratio = mean(trade_returns) / std(trade_returns) * sqrt(252) if std > 0, else 0.0. Use 252 annualization factor (trading days). If num_trades < 2, sharpe = 0.0.
       - pnl_series: cumulative P&L at each trade (for plotting equity curves).

    6. Create test infrastructure: tests/models/__init__.py, tests/evaluation/__init__.py, tests/evaluation/conftest.py.
       - conftest.py: fixture providing sample y_true and y_pred numpy arrays (10 elements each with known spread values including positive, negative, and near-zero).
