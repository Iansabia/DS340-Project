1. Create `src/data/kalshi.py` implementing `KalshiAdapter(MarketDataAdapter)`:

   Class attributes:
   - `BASE_URL = "https://api.elections.kalshi.com/trade-api/v2"`
   - `CATEGORIES = ["Economics", "Crypto", "Financials"]` (per locked decision)
   - `MAX_CANDLES_PER_REQUEST = 4500` (buffer below 5000 API limit)
   - `PERIOD_INTERVAL = 1` (minute-level per locked decision)

   Constructor `__init__(self, client: ResilientClient | None = None)`:
   - If client is None, create `ResilientClient(base_url=BASE_URL, max_retries=3, backoff_factor=1.0, requests_per_second=18.0)` (18 req/s leaves buffer below Kalshi's 20 req/s Basic tier limit)
   - Store as `self.client`

   Method `list_markets(self, categories: list[str] | None = None) -> list[MarketMetadata]`:
   - Default categories to `self.CATEGORIES` if None
   - For each category, call `self.client.get("series", params={"category": category})` to get series list
   - For each series, paginate `self.client.get("events", params={"series_ticker": series_ticker, "status": "settled", "with_nested_markets": True, "limit": 200, "cursor": cursor})` until cursor is empty string or absent
   - For each market in each event, create `MarketMetadata(market_id=market["ticker"], question=market.get("title", ""), category=category, platform="kalshi", resolution_date=market.get("close_time", ""), result=market.get("result"), outcomes=["Yes", "No"])`
   - Return full list of MarketMetadata objects
   - Log count per category

   Method `get_candlesticks(self, market_id: str, **kwargs) -> pd.DataFrame`:
   - Accept optional `start_ts` and `end_ts` in kwargs. If not provided, use sensible defaults: `start_ts = 0` (fetch from earliest available) and `end_ts = int(time.time())`
   - Compute `chunk_seconds = MAX_CANDLES_PER_REQUEST * PERIOD_INTERVAL * 60` (= 270000 seconds = ~3.125 days)
   - Loop: current_start from start_ts to end_ts, incrementing by chunk_seconds
   - Each iteration: `self.client.get(f"historical/markets/{market_id}/candlesticks", params={"start_ts": current_start, "end_ts": min(current_start + chunk_seconds, end_ts), "period_interval": PERIOD_INTERVAL})`
   - Collect all candlesticks from `data.get("candlesticks", [])`
   - Parse with `_parse_candlesticks(candles)` method
   - Return concatenated DataFrame sorted by timestamp, deduplicated

   Method `_parse_candlesticks(self, candles: list[dict]) -> pd.DataFrame`:
   - For each candlestick dict:
     - `timestamp = c["end_period_ts"]` (int, Unix seconds)
     - Extract `price = c.get("price", {})`; `trade_close = price.get("close")`
     - Extract `yes_bid = c.get("yes_bid", {})`; `bid_close = yes_bid.get("close")`
     - Extract `yes_ask = c.get("yes_ask", {})`; `ask_close = yes_ask.get("close")`
     - If `trade_close is not None`: `close = float(trade_close)`, `has_trades = True`
     - Elif `bid_close is not None and ask_close is not None`: `close = (float(bid_close) + float(ask_close)) / 2`, `has_trades = False`
     - Else: `close = None`, `has_trades = False`
     - Similarly for open/high/low: use `price.get("open")` etc., convert to float if not None
     - `volume = float(c.get("volume", "0"))`
     - `open_interest = float(c.get("open_interest", "0"))`
     - `yes_bid_close = float(bid_close) if bid_close is not None else None`
     - `yes_ask_close = float(ask_close) if ask_close is not None else None`
   - Return DataFrame with columns: timestamp, open, high, low, close, volume, open_interest, yes_bid_close, yes_ask_close, has_trades
   - Log: `f"Parsed {len(rows)} candlesticks, {sum(1 for r in rows if r['has_trades'])} with trades"`

1. Create `src/data/ingest_kalshi.py` as the CLI entry point:
   ```python
   """CLI script to ingest all Kalshi historical market data.
   
   Usage:
       .venv/bin/python -m src.data.ingest_kalshi [--categories Economics Crypto Financials] [--output-dir data/raw/kalshi]
   
   Discovers all resolved markets in the specified categories,
   fetches minute-level candlesticks, and saves as parquet files.
   Idempotent: re-running skips already-cached markets.
   """
   import argparse
   import json
   import logging
   from pathlib import Path

   from src.data.kalshi import KalshiAdapter

   logging.basicConfig(
       level=logging.INFO,
       format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
       datefmt="%Y-%m-%d %H:%M:%S",
   )
   logger = logging.getLogger(__name__)

   DEFAULT_CATEGORIES = ["Economics", "Crypto", "Financials"]
   DEFAULT_OUTPUT_DIR = Path("data/raw/kalshi")


   def main():
       parser = argparse.ArgumentParser(description="Ingest Kalshi historical market data")
       parser.add_argument(
           "--categories",
           nargs="+",
           default=DEFAULT_CATEGORIES,
           help=f"Market categories to ingest (default: {DEFAULT_CATEGORIES})",
       )
       parser.add_argument(
           "--output-dir",
           type=Path,
           default=DEFAULT_OUTPUT_DIR,
           help=f"Output directory for parquet files (default: {DEFAULT_OUTPUT_DIR})",
       )
       args = parser.parse_args()

       logger.info(f"Starting Kalshi ingestion: categories={args.categories}, output={args.output_dir}")
       args.output_dir.mkdir(parents=True, exist_ok=True)

       adapter = KalshiAdapter()
       
       # First, discover and save market metadata
       markets = adapter.list_markets(args.categories)
       metadata_path = args.output_dir / "_metadata.json"
       metadata_records = [
           {
               "market_id": m.market_id,
               "question": m.question,
               "category": m.category,
               "platform": m.platform,
               "resolution_date": m.resolution_date,
               "result": m.result,
               "outcomes": m.outcomes,
           }
           for m in markets
       ]
       with open(metadata_path, "w") as f:
           json.dump(metadata_records, f, indent=2)
       logger.info(f"Saved metadata for {len(markets)} markets to {metadata_path}")

       # Then, fetch candlesticks for each market (with caching)
       stats = adapter.ingest_all(args.categories, args.output_dir)
       
       logger.info(f"Ingestion complete: {json.dumps(stats, indent=2)}")
       return stats


   if __name__ == "__main__":
       main()
   ```

   The script:
   - Accepts --categories and --output-dir CLI arguments
   - Creates output directory if it does not exist
   - Discovers markets and saves metadata as `_metadata.json` (needed by Phase 2 matching)
   - Calls `adapter.ingest_all()` which uses caching (idempotent re-runs)
   - Logs progress and final statistics
