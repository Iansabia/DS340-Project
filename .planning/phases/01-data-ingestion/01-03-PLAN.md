<objective>
Build the Polymarket platform adapter that discovers resolved markets using keyword-based filtering via Gamma API, fetches price history from CLOB API (with Data API trade fallback), and writes parquet files to data/raw/polymarket/.

Purpose: Polymarket is the second prediction market platform. This adapter produces the raw Polymarket data that Phase 2 (Market Matching) needs to find cross-platform pairs. The Polymarket API is notably more complex than Kalshi's (three separate APIs with different quirks).

Output: PolymarketAdapter class implementing MarketDataAdapter, a CLI ingestion script, and comprehensive tests.
</objective>

1. Create `src/data/polymarket.py` implementing `PolymarketAdapter(MarketDataAdapter)`:

   Class attributes:
   - `GAMMA_BASE = "https://gamma-api.polymarket.com"`
   - `CLOB_BASE = "https://clob.polymarket.com"`
   - `DATA_BASE = "https://data-api.polymarket.com"`
   - `CHUNK_DAYS = 14` (chunk CLOB price history into 14-day windows)
   - `MAX_TRADE_OFFSET = 3000` (hard API limit for Data API pagination)
   - Keyword lists (per RESEARCH.md):
     ```python
     CRYPTO_KEYWORDS = [
         "btc", "bitcoin", "eth", "ethereum", "solana", "sol ",
         "crypto", "defi", "token", "altcoin", "nft",
     ]
     FINANCE_KEYWORDS = [
         "fed", "inflation", "cpi", "gdp", "recession", "unemployment",
         "rate cut", "rate hike", "interest rate", "s&p", "nasdaq",
         "dow", "treasury", "bond", "yield", "fomc",
     ]
     KEYWORD_MAP = {
         "crypto": CRYPTO_KEYWORDS,
         "finance": FINANCE_KEYWORDS,
     }
     ```

   Constructor `__init__(self, gamma_client=None, clob_client=None, data_client=None)`:
   - Create three separate ResilientClient instances (one per API):
     - `self.gamma_client = gamma_client or ResilientClient(base_url=GAMMA_BASE, max_retries=3, backoff_factor=1.0, requests_per_second=10.0)`
     - `self.clob_client = clob_client or ResilientClient(base_url=CLOB_BASE, max_retries=3, backoff_factor=1.0, requests_per_second=10.0)`
     - `self.data_client = data_client or ResilientClient(base_url=DATA_BASE, max_retries=3, backoff_factor=1.0, requests_per_second=10.0)`
   - Also store `self._market_token_map: dict[str, list[str]] = {}` to cache conditionId -> clobTokenIds mapping

   Method `list_markets(self, categories: list[str] | None = None) -> list[MarketMetadata]`:
   - Default categories to `list(self.KEYWORD_MAP.keys())` if None
   - Build combined keyword list from requested categories
   - Paginate Gamma API: `self.gamma_client.get("events", params={"closed": "true", "limit": 100, "offset": offset})`
   - For each event, check if any keyword from the combined list appears in `event["title"].lower()` or `event["description"].lower()` (joined as `f"{title} {description}"`)
   - Determine category by which keyword list matched (if crypto keyword matched -> "crypto", if finance keyword matched -> "finance")
   - For each matching market in the event:
     - Parse clobTokenIds: `json.loads(market.get("clobTokenIds", "[]"))` -- CRITICAL: these are stringified JSON arrays in Gamma responses
     - Store mapping: `self._market_token_map[conditionId] = clob_token_ids`
     - Create `MarketMetadata(market_id=market["conditionId"], question=market.get("question", ""), category=detected_category, platform="polymarket", resolution_date=market.get("endDate", ""), result=None, outcomes=json.loads(market.get("outcomes", '["Yes", "No"]')) if isinstance(market.get("outcomes"), str) else market.get("outcomes", ["Yes", "No"]))`
   - Stop paginating when response returns fewer than 100 events
   - Return list of MarketMetadata

   Method `get_candlesticks(self, market_id: str, **kwargs) -> pd.DataFrame`:
   - `market_id` is a conditionId
   - Look up clobTokenIds from `self._market_token_map.get(market_id, [])`. If empty, log warning and return empty DataFrame.
   - **Primary strategy: CLOB /prices-history with startTs/endTs**
     - Try clobTokenIds[0] (Yes token) first
     - Compute start_ts and end_ts: `start_ts = kwargs.get("start_ts", 0)`, `end_ts = kwargs.get("end_ts", int(time.time()))`
     - Chunk into CHUNK_DAYS windows (14 * 86400 seconds each)
     - For each chunk: `self.clob_client.get("prices-history", params={"market": token_id, "startTs": current_start, "endTs": current_end, "fidelity": 60})`
     - Parse response: history = data.get("history", []). Each point is {"t": timestamp_int, "p": price_string}
     - If Yes token returned no data, try clobTokenIds[1] (No token) if it exists
     - Convert to DataFrame: rename "t" -> "timestamp", "p" -> "price". Set `close = float(price)`. Set open/high/low = close (CLOB only provides single price per interval). Set volume = 0 (CLOB does not provide volume in price history).
   - **Fallback strategy: Data API /trades** (only if CLOB returned no data for either token)
     - `self._fetch_trades(market_id)` fetches trades with offset pagination up to MAX_TRADE_OFFSET
     - `self._trades_to_ohlcv(trades, freq_minutes=60)` aggregates into hourly bars
   - Deduplicate by timestamp, sort ascending
   - Return DataFrame with CANDLESTICK_COLUMNS + any optional columns

   Method `_fetch_trades(self, condition_id: str) -> list[dict]`:
   - Paginate Data API: `self.data_client.get("trades", params={"market": condition_id, "limit": 500, "offset": offset})`
   - Stop at offset > MAX_TRADE_OFFSET (3000) or when fewer than 500 trades returned
   - Return all collected trades

   Method `_trades_to_ohlcv(self, trades: list[dict], freq_minutes: int = 60) -> pd.DataFrame`:
   - Create DataFrame from trades list
   - Convert timestamp to datetime UTC
   - Convert price and size to float
   - Resample to `freq_minutes` intervals:
     - open = first price, high = max, low = min, close = last
     - volume = sum of sizes
   - Drop any intervals with zero trades (NaN)
   - Reset index, convert timestamp back to Unix seconds int
   - Return DataFrame with CANDLESTICK_COLUMNS

1. Create `src/data/ingest_polymarket.py` as the CLI entry point:
   ```python
   """CLI script to ingest all Polymarket historical market data.
   
   Usage:
       .venv/bin/python -m src.data.ingest_polymarket [--categories crypto finance] [--output-dir data/raw/polymarket]
   
   Discovers resolved markets via Gamma API keyword filtering,
   fetches price history from CLOB API (with Data API trade fallback),
   and saves as parquet files. Idempotent: re-running skips already-cached markets.
   """
   import argparse
   import json
   import logging
   from pathlib import Path

   from src.data.polymarket import PolymarketAdapter

   logging.basicConfig(
       level=logging.INFO,
       format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
       datefmt="%Y-%m-%d %H:%M:%S",
   )
   logger = logging.getLogger(__name__)

   DEFAULT_CATEGORIES = ["crypto", "finance"]
   DEFAULT_OUTPUT_DIR = Path("data/raw/polymarket")


   def main():
       parser = argparse.ArgumentParser(description="Ingest Polymarket historical market data")
       parser.add_argument(
           "--categories",
           nargs="+",
           default=DEFAULT_CATEGORIES,
           help=f"Market categories to ingest (default: {DEFAULT_CATEGORIES})",
       )
       parser.add_argument(
           "--output-dir",
           type=Path,
           default=DEFAULT_OUTPUT_DIR,
           help=f"Output directory for parquet files (default: {DEFAULT_OUTPUT_DIR})",
       )
       args = parser.parse_args()

       logger.info(f"Starting Polymarket ingestion: categories={args.categories}, output={args.output_dir}")
       args.output_dir.mkdir(parents=True, exist_ok=True)

       adapter = PolymarketAdapter()
       
       # First, discover and save market metadata (includes clobTokenIds mapping)
       markets = adapter.list_markets(args.categories)
       
       # Save metadata including the token ID mapping for downstream use
       metadata_path = args.output_dir / "_metadata.json"
       metadata_records = [
           {
               "market_id": m.market_id,
               "question": m.question,
               "category": m.category,
               "platform": m.platform,
               "resolution_date": m.resolution_date,
               "result": m.result,
               "outcomes": m.outcomes,
               "clob_token_ids": adapter._market_token_map.get(m.market_id, []),
           }
           for m in markets
       ]
       with open(metadata_path, "w") as f:
           json.dump(metadata_records, f, indent=2)
       logger.info(f"Saved metadata for {len(markets)} markets to {metadata_path}")

       # Then, fetch price history for each market (with caching)
       stats = adapter.ingest_all(args.categories, args.output_dir)
       
       # Log data source breakdown
       logger.info(f"Ingestion complete: {json.dumps(stats, indent=2)}")
       return stats


   if __name__ == "__main__":
       main()
   ```

   The script:
   - Accepts --categories (crypto, finance) and --output-dir CLI arguments
   - Creates output directory if it does not exist
   - Discovers markets via keyword filtering and saves metadata as `_metadata.json`
   - Includes `clob_token_ids` in metadata (needed by Phase 2 matching for reference)
   - Calls `adapter.ingest_all()` which handles CLOB-first/Data-API-fallback and caching
   - Logs progress and final statistics
