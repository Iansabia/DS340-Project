
Transaction cost model:
- Each trade incurs a cost of `cost_pp` (percentage points) deducted from P&L
- net_pnl = total_pnl - (num_trades * cost_pp)
- Break-even cost = total_pnl / num_trades (the per-trade cost that zeroes out P&L)

From existing results JSON schema:
```python
{"metrics": {"total_pnl": float, "num_trades": int, ...}}
```
