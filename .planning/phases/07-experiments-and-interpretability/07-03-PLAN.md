<objective>
Run Experiment 3: spread threshold ablation. Re-evaluate ALL 8 trained models with trading thresholds {0.0, 0.02, 0.05, 0.10}. No retraining -- only the profit simulation threshold changes. This tests how the minimum spread threshold affects trading frequency and profitability.

Purpose: Answers whether a higher threshold (more selective trading) improves risk-adjusted returns. At threshold=0.0 all predictions trigger trades; at threshold=0.10 only large predicted moves trigger trades.
Output: 32 result JSONs (8 models x 4 thresholds), P&L heatmap, grouped bar chart.
</objective>
