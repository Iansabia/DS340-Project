<objective>
Compute bootstrap confidence intervals (1000 resamples, 95% CI) on key metrics for all 8 models. This adds statistical rigor to the cross-tier comparison: if XGBoost's RMSE CI overlaps with GRU's, the difference isn't statistically significant.

Purpose: Without CIs, we can't distinguish real performance differences from sampling noise. Bootstrap resampling of test predictions gives nonparametric CIs without distributional assumptions.
Output: CI JSON, formatted CI table, RMSE forest plot.
</objective>

3. **Bootstrap loop** (vectorized where possible):
   ```python
   rng = np.random.RandomState(SEED)
   n = len(y_test)
   
   # Pre-generate all bootstrap indices at once for reproducibility
   all_indices = rng.choice(n, size=(N_BOOTSTRAP, n), replace=True)
   
   for model_name, predictions in model_predictions.items():
       boot_metrics = {metric: [] for metric in ["rmse", "mae", "directional_accuracy", "total_pnl", "num_trades", "win_rate", "sharpe_ratio"]}
       
       for b in range(N_BOOTSTRAP):
           idx = all_indices[b]
           y_boot = y_test[idx]
           p_boot = predictions[idx]
           ts_boot = test_timestamps[idx] if test_timestamps is not None else None
           
           reg = compute_regression_metrics(y_boot, p_boot)
           trd = simulate_profit(p_boot, y_boot, threshold=THRESHOLD, timestamps=ts_boot)
           
           for k in boot_metrics:
               boot_metrics[k].append(reg.get(k, trd.get(k, 0.0)))
       
       # Compute CIs
       ci_lower = {k: float(np.percentile(v, 2.5)) for k, v in boot_metrics.items()}
       ci_upper = {k: float(np.percentile(v, 97.5)) for k, v in boot_metrics.items()}
       ci_mean = {k: float(np.mean(v)) for k, v in boot_metrics.items()}
   ```
