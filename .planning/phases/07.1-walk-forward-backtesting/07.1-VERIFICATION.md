### Notable Findings

**Sharpe range is higher than typical financial markets but consistent with the data:**
Backtested Sharpe values run from -2.26 (Naive) to 8.70 (Linear Regression). The SUMMARY documents a deliberate decision to accept this range as valid: fees are deducted, capital is normalized, and naive baselines correctly go negative. The high absolute values reflect the academic-quality-filtered dataset and 24/7 market structure (sqrt(365) annualization). No inflated 15-23 values remain.

**break_even_fee_pp is at JSON top level, not under "metrics":**
The field is present in all 8 JSON files (e.g., XGBoost: 0.1748, meaning 17.48pp round-trip break-even). It is stored via `save_results(..., extra={"break_even_fee_pp": ...})` which merges it at the top-level payload rather than under `metrics`. This is correct behavior per results_store.py but means code reading `r['metrics']['break_even_fee_pp']` would fail — must read `r['break_even_fee_pp']`. The run_backtest.py `format_backtest_table` reads it correctly at line 608.
