Create `experiments/run_backtest.py` that runs all 8 models through the WalkForwardBacktester. Follow the same patterns as `run_baselines.py` for data loading and model instantiation.

**Script structure:**

1. **Imports and constants:**
   - Import `load_train_test, _build_split, _feature_columns, prepare_xy, prepare_xy_for_seq, NON_FEATURE_COLUMNS` from `experiments.run_baselines`
   - Import `WalkForwardBacktester, compute_break_even_fee` from `src.evaluation.backtester`
   - Import `save_results, load_all_results` from `src.evaluation.results_store`
   - Import all model classes (same as run_baselines.py imports)
   - `BACKTEST_RESULTS_DIR = Path("experiments/results/backtest")`
   - `FIGURES_DIR = Path("experiments/figures")`

2. **`run_single_model_backtest(model, df_test, predictions, backtester) -> dict`:**
   - Calls `backtester.run(df_test, predictions)`
   - Returns the result dict

3. **`run_all_backtests(tier="all", threshold=0.02) -> list[dict]`:**
   - Load train/test data using `load_train_test` + `_build_split`
   - Get feature columns via `_feature_columns`
   - Instantiate WalkForwardBacktester with CONTEXT.md defaults ($10k capital, $100/position, 3pp entry, 2pp exit)
   - For each model tier:
     - **Tier 1 (Naive, Volume, LinReg, XGBoost):** fit on train, predict on test using `prepare_xy`
     - **Tier 2 (GRU, LSTM):** fit on train, predict on test using `prepare_xy_for_seq`. Use seed=42 only (single backtest, not 3-seed -- we want one equity curve per model).
     - **Tier 3 (PPO-Raw, PPO-Filtered):** Same as Tier 2 pattern. For PPO-Filtered, train one autoencoder (seed=42) then PPO. Use seed=42.
   - For each model: run backtest, also compute break_even_fee
   - Save each result to `experiments/results/backtest/{model_slug}.json` via save_results
   - The JSON should include standard metrics fields PLUS backtest-specific fields: `max_drawdown`, `calmar_ratio`, `total_fees`, `break_even_fee_pp`, `equity_curve`, `daily_returns`
   - Return list of all result dicts

4. **`plot_equity_curves(results: list[dict], output_path: Path)`:**
   - Use matplotlib with tier-distinct styling:
     - Tier 1 (Naive, Volume, LinReg, XGBoost): solid lines, muted colors
     - Tier 2 (GRU, LSTM): dashed lines, brighter colors
     - Tier 3 (PPO-Raw, PPO-Filtered): dotted lines
   - X-axis: trading day number (not timestamp for simplicity)
   - Y-axis: portfolio value ($)
   - Horizontal dashed line at $10,000 (initial capital)
   - Title: "Walk-Forward Backtest: Equity Curves ($10k initial, 5pp round-trip costs)"
   - Legend with model names
   - Save to `experiments/figures/backtest_equity_curves.png` at 300 DPI
   - Use `plt.close()` after saving (no display)

5. **`plot_drawdown(results: list[dict], output_path: Path)`:**
   - Compute drawdown series from each model's equity_curve
   - Plot as filled areas (negative values below 0)
   - Title: "Walk-Forward Backtest: Drawdown"
   - Save to `experiments/figures/backtest_drawdown.png` at 300 DPI

6. **`format_backtest_table(results: list[dict]) -> str`:**
   - Columns: Model | Sharpe (BT) | Old Sharpe | Total P&L | Max DD | Calmar | Win Rate | Trades | Fees | Break-Even Fee
   - "Old Sharpe" loaded from existing tier1/tier2/tier3 result JSONs for side-by-side comparison
   - Fixed model display order: Naive, Volume, LinReg, XGBoost, GRU, LSTM, PPO-Raw, PPO-Filtered

7. **`main()` with argparse:**
   - `--tier`: choices ["1", "2", "3", "all"], default "all"
   - `--threshold`: float, default 0.02
   - `--data-dir`: Path, default `data/processed`
   - Runs backtests, generates figures, prints table
   - Entry: `python -m experiments.run_backtest`

Use `torch.set_num_threads(1)` at top of main (same Apple Silicon workaround as run_baselines).
