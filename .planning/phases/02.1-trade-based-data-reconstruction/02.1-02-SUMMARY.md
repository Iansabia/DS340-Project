## What Was Built

### Cross-Platform Aligner (`src/data/aligner.py`)
- Unified 4-hour time grid creation from market lifetime overlap
- Forward-fill with limit=6 bars (24 hours) per platform
- `hours_since_last_trade` computation per platform per bar
- Spread calculation: `kalshi_vwap - polymarket_vwap`
- Quality filters: min 20 trades, staleness <80%, mean |spread| <0.30

### Pipeline CLI (`scripts/rebuild_data.py`)
- 4-stage orchestration: load pairs → fetch trades → reconstruct candles → align + validate
- Outputs: `data/processed/aligned_pairs.parquet` + `data/processed/data_quality_report.json`
- Full pipeline runs in 98.6 seconds
