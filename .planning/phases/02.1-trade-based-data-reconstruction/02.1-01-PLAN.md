**Create src/data/trade_reconstructor.py** with one public function:

`reconstruct_candles(trades_df: pd.DataFrame, bar_seconds: int = 14400) -> pd.DataFrame`:
- Input: DataFrame with columns ["timestamp", "price", "volume", "side"] (normalized trade format from trade_fetcher)
- If trades_df is empty, return empty DataFrame with CANDLE_COLUMNS
- Clip prices to [0.01, 0.99]: `trades_df["price"] = trades_df["price"].clip(0.01, 0.99)`
- Compute bar timestamps: `trades_df["bar_ts"] = (trades_df["timestamp"] // bar_seconds) * bar_seconds`
- Group by bar_ts, for each group compute:
  - `vwap = (group["price"] * group["volume"]).sum() / group["volume"].sum()`
  - `open = group["price"].iloc[0]` (first trade, sorted by timestamp)
  - `high = group["price"].max()`
  - `low = group["price"].min()`
  - `close = group["price"].iloc[-1]` (last trade)
  - `volume = group["volume"].sum()`
  - `trade_count = len(group)`
  - `dollar_volume = (group["price"] * group["volume"]).sum()`
  - `buy_volume = group.loc[group["side"] == "buy", "volume"].sum()`
  - `sell_volume = group.loc[group["side"] == "sell", "volume"].sum()`
  - `realized_spread`: compute `sell_mean = group.loc[group["side"]=="sell", "price"].mean()` and `buy_mean = group.loc[group["side"]=="buy", "price"].mean()`. If both exist: `sell_mean - buy_mean`. If only one side: `0.0`. Use `float("nan")` only if zero trades (which can't happen since we only emit bars with trades).
  - `max_trade_size = group["volume"].max()`
  - `has_trade = True`
- Timestamp column = bar_ts (start of 4-hour window)
- Sort by timestamp ascending, reset index
- Return DataFrame with columns in CANDLE_COLUMNS order

Note: `time_since_last_trade` is NOT computed here -- it is computed in the aligner (Plan 02) because it requires cross-bar context on the unified grid.
