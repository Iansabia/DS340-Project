Create `src/models/trading_env.py` with class `SpreadTradingEnv(gymnasium.Env)`.

Implementation per CONTEXT.md locked decisions:

**Constructor:**
- Accept X_train (DataFrame with group_id), y_train (spread change targets), feature_cols (list of 31 column names), lookback=6, transaction_cost=0.02, reward_clip=0.5, scaler (optional pre-fitted StandardScaler)
- Extract group_ids from X_train["group_id"]
- If scaler is None, call `fit_feature_scaler()` from sequence_utils on X_train[feature_cols] (with bool_cols detection)
- Scale features via `apply_feature_scaler()`
- Group data by group_id into list of per-pair arrays: each is (n_bars, 31) scaled features + (n_bars,) targets
- Filter out pairs where n_bars < lookback (skip short pairs, per CONTEXT.md)
- Set observation_space = gymnasium.spaces.Box(low=-np.inf, high=np.inf, shape=(lookback * 33,), dtype=np.float32)
- Set action_space = gymnasium.spaces.Discrete(3)

**reset(seed=None, options=None):**
- Advance to next pair (cycle through pairs list)
- Set current_step = lookback - 1 (first valid step after lookback window)
- Set current_position = 0 (no position)
- Return flattened observation of shape (198,) and info dict

**step(action):**
- Map action: {0: 0 (hold), 1: +1 (long), 2: -1 (short)}
- Compute position_change = abs(new_position - current_position)
- Compute reward = current_position * y_train[current_step] - transaction_cost * position_change
- Clip reward to [-reward_clip, +reward_clip]
- Update current_position
- Advance current_step
- terminated = current_step >= len(pair_data)
- Build next observation (or zeros if terminated)
- Return (obs, reward, terminated, False, info)

**_get_observation():**
- Extract lookback window of scaled features: pair_features[current_step - lookback + 1 : current_step + 1]  shape (lookback, 31)
- Augment each row with current_position and time_fraction = current_step / (len(pair) - 1)
- Flatten to (198,) = (6, 33).flatten()
- Return as float32

**Important:** Do NOT inherit from BasePredictor. This is an environment, not a predictor.

Import from sequence_utils: fit_feature_scaler, apply_feature_scaler
