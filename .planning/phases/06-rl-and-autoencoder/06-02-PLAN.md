Implementation per CONTEXT.md locked decisions:

**Internal _AutoencoderModule(nn.Module):**
- Encoder: Linear(31->16) -> ReLU -> BatchNorm1d(16) -> Linear(16->8) -> ReLU -> BatchNorm1d(8) -> Linear(8->4)
- Decoder: Linear(4->8) -> ReLU -> BatchNorm1d(8) -> Linear(8->16) -> ReLU -> BatchNorm1d(16) -> Linear(16->31)
- Forward returns reconstruction and bottleneck separately (for potential future use)

**AnomalyDetectorAutoencoder:**

Constructor: input_dim=31, bottleneck_dim=4, lr=1e-3, batch_size=32, max_epochs=200, patience=20, min_delta=1e-4, random_state=42

**fit(X_train, feature_cols):**
- Extract feature columns, detect bool_cols, fit scaler via `fit_feature_scaler()`
- Scale features via `apply_feature_scaler()`
- 90/10 chronological val split (last 10% of rows)
- Build _AutoencoderModule, Adam optimizer lr=1e-3, MSE loss
- Training loop: shuffle train, batch_size=32, early stopping patience=20 min_delta=1e-4
- After training, call `set_threshold(X_train)` automatically
- Return self

**compute_reconstruction_error(X):**
- Scale using fitted scaler
- Forward pass through model
- Return per-sample MSE: mean((input - reconstruction)^2, axis=1) as 1D ndarray

**set_threshold(X_train, percentile=95.0):**
- Compute reconstruction errors on training data
- Set self.threshold_ = np.percentile(errors, percentile)

**flag_anomalies(X):**
- Compute reconstruction errors
- Return boolean array: errors > self.threshold_

Import from sequence_utils: fit_feature_scaler, apply_feature_scaler, EarlyStopping, set_seed, get_device
