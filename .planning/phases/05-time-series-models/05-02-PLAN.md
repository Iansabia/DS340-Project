<objective>
Implement GRUPredictor — the first Tier 2 recurrent model. Inherits BasePredictor (same contract as Tier 1), caches training data during fit(), builds 6-bar lookback windows internally with warm-up stitching during predict() so it returns one prediction per input row (row-aligned with XGBoost for direct comparison).

Purpose: MOD-05 requirement. Tests whether temporal structure (24-hour lookback on 4-hour bars) improves spread-change prediction over the flat-tabular Tier 1 baseline. GRU is preferred over LSTM per CLAUDE.md (simpler, faster, works well on short sequences).

Output: `src/models/gru.py` with GRUPredictor class, fully tested via `tests/models/test_gru.py` (RED-GREEN TDD cycle).

    **PyTorch module (internal class `_GRUModule(nn.Module)`):**
    ```python
    class _GRUModule(nn.Module):
        def __init__(self, n_features, hidden_size, num_layers, dropout):
            super().__init__()
            self.dropout = nn.Dropout(dropout)
            self.gru = nn.GRU(input_size=n_features, hidden_size=hidden_size,
                              num_layers=num_layers, batch_first=True, bidirectional=False)
            self.out = nn.Linear(hidden_size, 1)
        def forward(self, x):  # x: (batch, lookback, n_features)
            x = self.dropout(x)
            h, _ = self.gru(x)
            last = h[:, -1, :]   # use last timestep hidden state
            return self.out(last).squeeze(-1)  # (batch,)
    ```

    **GRUPredictor class methods (signatures from `<interfaces>` block above):**

    `__init__`: Store all hyperparameters as private attributes (`_hidden_size`, `_num_layers`, `_dropout`, `_learning_rate`, `_weight_decay`, `_batch_size`, `_max_epochs`, `_patience`, `_min_delta`, `_lookback=6`, `_grad_clip=1.0`, `_random_state=42`). Initialize `self._fitted = False`, `self._model = None`, `self._scaler = None`, `self._cached_train = None`.

    `name` property returns `"GRU"`.

    `fit(X_train, y_train)`:
    1. Call `set_seed(self._random_state)` from sequence_utils
    2. Identify feature columns: `feature_cols = [c for c in X_train.columns if c != "group_id"]`
    3. Identify bool columns by dtype: `bool_cols = [c for c in feature_cols if X_train[c].dtype == bool]`
    4. Fit scaler: `self._scaler = fit_feature_scaler(X_train[feature_cols], bool_cols)`
    5. Extract group_ids: `group_ids = X_train["group_id"].to_numpy()`
    6. Apply scaler: `X_scaled = apply_feature_scaler(X_train[feature_cols], self._scaler, bool_cols)`
    7. 90/10 within-pair chronological val split (per CONTEXT.md D4): for each unique group_id, last 10% of rows (by row order) → val, rest → train. Use np.unique with return_index and slicing.
    8. Build windows via `create_sequences(X_scaled_train, y_train_split, self._lookback, group_ids_train)` for both train and val.
    9. Convert to torch tensors on `get_device()`.
    10. Initialize `_GRUModule(n_features=len(feature_cols), hidden_size=self._hidden_size, num_layers=self._num_layers, dropout=self._dropout).to(device)`. Store as `self._model`.
    11. Optimizer: `torch.optim.AdamW(self._model.parameters(), lr=self._learning_rate, weight_decay=self._weight_decay)`
    12. Scheduler: `torch.optim.lr_scheduler.ReduceLROnPlateau(optimizer, mode='min', factor=0.5, patience=5)`
    13. Loss: `nn.MSELoss()`
    14. EarlyStopping(patience=self._patience, min_delta=self._min_delta)
    15. Training loop (max_epochs):
        - For each epoch: shuffle train indices, iterate batch_size batches, forward, loss, backward, `torch.nn.utils.clip_grad_norm_(self._model.parameters(), max_norm=self._grad_clip)`, step.
        - Compute val_loss on val windows (no grad), step scheduler, check EarlyStopping.
        - If no val windows (too-short dataset), skip val — run full max_epochs.
        - Print or log `val_loss` per epoch.
    16. Cache training data for warm-up stitching: `self._cached_train = {"X_scaled": X_scaled, "group_ids": group_ids, "feature_cols": feature_cols, "bool_cols": bool_cols}`
    17. `self._fitted = True`; `return self`.

    `predict(X)`:
    1. If not self._fitted: `raise RuntimeError("Model must be fit before predict")`
    2. Identify feature_cols from cached_train. Apply scaler to X[feature_cols]: `X_scaled = apply_feature_scaler(...)`.
    3. Warm-up stitching (per CONTEXT.md D2): for each group_id in X, concatenate cached_train rows of same group_id (if any) + X rows of that group, IN ORDER. Build windows on the concatenated sequence.
    4. For each row in X (in order), extract the window ending at that row from the stitched sequence. If the group has fewer than lookback total rows available (train+test), pad at front by repeating the first available row.
    5. Batch forward through self._model in eval mode with torch.no_grad(). Collect predictions.
    6. Return np.ndarray shape (len(X),) of dtype float.
