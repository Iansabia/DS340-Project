<objective>
Implement GRUPredictor â€” the first Tier 2 recurrent model. Inherits BasePredictor (same contract as Tier 1), caches training data during fit(), builds 6-bar lookback windows internally with warm-up stitching during predict() so it returns one prediction per input row (row-aligned with XGBoost for direct comparison).

Purpose: MOD-05 requirement. Tests whether temporal structure (24-hour lookback on 4-hour bars) improves spread-change prediction over the flat-tabular Tier 1 baseline. GRU is preferred over LSTM per CLAUDE.md (simpler, faster, works well on short sequences).

Output: `src/models/gru.py` with GRUPredictor class, fully tested via `tests/models/test_gru.py` (RED-GREEN TDD cycle).

BasePredictor exports (inherited, do NOT override):
- `evaluate(X_test, y_test, threshold=0.02) -> dict` â€” calls self.predict, combines regression_metrics + profit_sim

**group_id contract:** Because `run_baselines.py` strips `group_id` via `NON_FEATURE_COLUMNS` before calling `.fit()`/`.predict()` on Tier 1 models, Tier 2 models receive an AUGMENTED DataFrame `X = df[feature_cols + ["group_id"]]`.    
    ```python
    class _GRUModule(nn.Module):
        def __init__(self, n_features, hidden_size, num_layers, dropout):
            super().__init__()
            self.dropout = nn.Dropout(dropout)
            self.gru = nn.GRU(input_size=n_features, hidden_size=hidden_size,
                              num_layers=num_layers, batch_first=True, bidirectional=False)
            self.out = nn.Linear(hidden_size, 1)
        def forward(self, x):  # x: (batch, lookback, n_features)
            x = self.dropout(x)
            h, _ = self.gru(x)
            last = h[:, -1, :]   # use last timestep hidden state
            return self.out(last).squeeze(-1)  # (batch,)
    ```
